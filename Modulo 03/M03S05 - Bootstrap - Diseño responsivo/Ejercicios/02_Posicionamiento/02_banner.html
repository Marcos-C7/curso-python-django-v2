Ejercicio

Genera una web sencilla que solo contenga algo parecido a lo que se muestra en la imágen:

  • El elemento pequeño contiene un padding tamaño 4, y el texto es un display-6.
  • El elemento grande contiene un ancho del 75% y un padding de tamaño 5.
  • El texto del elemento grande tiene un padding superior de tamaño 3.
